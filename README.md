# Chrome extension QuickTube-Blocker
Youtube Ad-Blocker. This script looks for the "Skip Ad" button because it wants to click it for you. It also finds the ad overlay and hides it quickly. Rather than waiting for the timer, we can set the video playback speed to 16x. This makes a long ad vanish in a blink.

## Installing Your Extension: The Local Way

### Prepare Your Files
Put your `manifest.json` and `content.js` into one folder on your desktop. Because Chrome needs to read the whole directory, you should name the folder something clear. Therefore, you can find it easily when you open your browser settings. While you might want to zip the folder, keep it unzipped. Rather than hiding it in your downloads, place it where it stays safe.

### Open Chrome Extensions
Navigate to `chrome://extensions/` by typing it into your address bar. Because this is a developer tool, you must toggle "Developer mode" to the "on" position. Therefore, the menu for loading local files will finally appear. While the screen looks busy, just look for the blue bar at the top. Rather than clicking "Update," you will focus on the loading button.

### Load the Folder
Click the button labeled "Load unpacked" in the top-left corner. Because a window will pop up, you should select your new folder. Therefore, Chrome will read your files and add the icon to your list. While it might show a warning about local scripts, you can ignore it safely. Rather than restarting the browser, you can just go to YouTube.

### Testing the Script
Refresh any open YouTube tabs to let the code take over the page. Because the script runs every half-second, it should catch ads before they even start. Therefore, you will see the video speed up and skip automatically. While it works silently, you can check the extension icon to see if it is active. Rather than waiting for a commercial, you can now enjoy your content.
